# 04 - 图表组件体系

## 1. 概述

MyQuant 前端包含两类图表渲染方案:

- **lightweight-charts (TradingView)**: 用于 K 线图、资金曲线, 面向金融数据的专业图表库
- **Recharts**: 用于分布图、趋势图, 面向通用数据可视化的 React 图表库

所有图表统一使用暗色主题, 与全局设计系统保持一致。

## 2. KlineChart (K 线图)

### 2.1 基本信息

文件位置: `src/components/charts/KlineChart.tsx`
依赖库: lightweight-charts (TradingView 开源版)
导出类型: `KlineChartHandle` (命令式接口)

### 2.2 Props 定义

```typescript
interface KlineChartProps {
  data: CandleData[];               // K 线数据
  indicators?: IndicatorData[];     // 叠加指标 (EMA 等)
  signals?: SignalData[];           // 买卖信号标记
  triggers?: TriggerMarker[];       // 触发点标记
  height?: number;                  // 图表高度 (默认 450)
  onTriggerClick?: (id: number) => void;  // 触发点点击回调
  bollData?: BollingerBandData;     // 布林带数据
  volumeData?: VolumeData[];        // 成交量数据
  mtfIndicators?: MTFIndicatorData[]; // 跨周期指标
}
```

### 2.3 命令式接口 (KlineChartHandle)

通过 `forwardRef` + `useImperativeHandle` 暴露:

```typescript
interface KlineChartHandle {
  scrollToTime: (time: string) => void;  // 滚动到指定时间
  zoomIn: () => void;                     // 放大 (可见范围缩小 20%)
  zoomOut: () => void;                    // 缩小 (可见范围扩大 25%)
  resetView: () => void;                  // 重置为全范围视图
}
```

缩放实现: 获取当前可见逻辑范围, 计算中心点, 按比例调整范围宽度。

### 2.4 图表结构

KlineChart 内部分为两个子图表:

```
+-------------------------------------+
| 主图区域 (78% 高度)                   |
|  - K 线 (CandlestickSeries)          |
|  - EMA 指标线 (LineSeries)            |
|  - 布林带 (3 条 LineSeries)           |
|  - 成交量柱 (HistogramSeries)         |
|  - 跨周期指标 (LineSeries)            |
|  - 买卖信号标记 (Markers)             |
|  - 触发点标记 (Markers on offset)     |
|  - 图例 (ChartEmbeddedLegend)         |
+-------------------------------------+
| RSI 子图 (22% 高度)                   |
|  - RSI 线 (LineSeries)               |
|  - 超买线 70 (虚线)                   |
|  - 超卖线 30 (虚线)                   |
+-------------------------------------+
```

时间轴同步: 使用 `useChartSync` hook 同步主图和 RSI 子图的时间范围。

### 2.5 时间转换

所有图表数据需要将字符串时间戳转换为 lightweight-charts 的 `UTCTimestamp`:

```typescript
function toTime(ts: string): Time {
  const withT = ts.includes("T") ? ts : ts.replace(" ", "T");
  const date = new Date(withT);
  if (isNaN(date.getTime())) {
    return ts.slice(0, 10) as Time;  // 回退到日期字符串
  }
  return Math.floor(date.getTime() / 1000) as UTCTimestamp;
}
```

K 线数据额外处理: 过滤无效时间 + 按时间升序排序。

### 2.6 系列管理

图表包含多种 Series, 通过 Ref 管理:

| Ref 名称               | Series 类型        | 用途              |
|------------------------|-------------------|-------------------|
| candleSeriesRef        | CandlestickSeries | K 线              |
| indicatorSeriesRefs    | Map<string, LineSeries> | 指标线    |
| bollSeriesRefs         | LineSeries[]      | 布林带 (上/中/下) |
| volumeSeriesRef        | HistogramSeries   | 成交量            |
| mtfSeriesRefs          | LineSeries[]      | 跨周期指标        |
| rsiSeriesRef           | LineSeries        | RSI 线            |
| triggerOffsetSeriesRef | LineSeries        | 触发标记载体      |

系列生命周期:

1. `useLayoutEffect`: 创建图表和基础系列 (K 线 + RSI), 注册 ResizeObserver
2. `useEffect` (数据更新): 增量更新系列数据, 移除过期系列, 添加新系列

### 2.9 触发点标记

触发标记通过独立的透明 LineSeries 实现, 偏移到 K 线高点上方:

```typescript
const TRIGGER_OFFSET = 1.012;  // 高点 * 1.012
```

| 触发结果 | 颜色     | 形状    | 文字        |
|---------|---------|---------|-------------|
| matched | #00C853 | circle  | (id)        |
| mismatched | #EF4444 | circle | (id)        |

### 2.10 布林带 (Bollinger Bands)

三条线:

- 上轨 (boll_upper): 实线, bollColor
- 中轨 (boll_middle): 虚线, bollColor + 50% 透明度
- 下轨 (boll_lower): 实线, bollColor

### 2.11 成交量

使用 HistogramSeries, 价格比例尺独立 (`priceScaleId: "vol"`):

- 占用主图底部 20% (`scaleMargins: { top: 0.8, bottom: 0 }`)
- 上涨 K 线对应成交量: `rgba(0,200,83,0.2)`
- 下跌 K 线对应成交量: `rgba(255,23,68,0.2)`

### 2.12 初始化与价格范围

首次数据加载时:

1. `fitContent()` 调整时间轴到全范围
2. 计算 K 线的最低价/最高价
3. 添加 8% 边距
4. 设置右侧价格轴的可见范围

autoScale 默认关闭, 防止用户缩放后价格范围被重置。

### 2.13 图例系统

文件位置: `src/components/charts/ChartEmbeddedLegend.tsx`

图例分组:

| 分组 ID  | 标签   | 内容                          |
|---------|--------|-------------------------------|
| ohlcv   | OHLCV  | K 线                          |
| ema     | 趋势   | 启用的 EMA 线                  |
| boll    | BOLL   | 上/中/下轨                     |
| mtf     | 跨周期 | 跨周期指标线                    |
| triggers| 触发点  | 触发标记数量                    |

交互:

- 十字线移动时实时更新各指标数值
- 点击图例项可切换对应系列的显示/隐藏 (设置 color 为 transparent 恢复)

### 2.14 全屏模式

KlineChart 内置全屏支持:

- 通过 `document.fullscreenElement` 检测状态
- 全屏时高度切换为 `100vh`
- 使用 `useState(isFullscreen)` 管理状态

## 11. 设计约束

1. KlineChart 使用 `forwardRef` 暴露命令式接口, 不通过 Props 回调控制缩放/定位
2. lightweight-charts 使用动态 import, 确保图表库不影响首屏加载
3. 触发标记使用透明偏移 Series 实现, 避免与 K 线系列冲突
4. autoScale 关闭, 首次加载手动计算价格范围, 用户可自由缩放而不被重置
5. 所有图表统一使用 DARK_CHART_THEME, 不支持主题切换
6. ResizeObserver 监听容器尺寸变化, 图表宽度自动适配
7. 数据更新时增量处理: 移除过期 Series, 保留未变化的 Series
