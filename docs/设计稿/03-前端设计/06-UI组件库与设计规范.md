# 06 - UI 组件库与设计规范

## 1. 概述

MyQuant 前端采用深色主题设计, 以 slate (石板灰) 为主色调, amber (琥珀金) 为强调色。
基础 UI 组件基于 Radix UI 原语构建, 图标库使用 Lucide React, 动画使用 Framer Motion。

## 3. 基础组件

### 3.1 GlassCard (玻璃卡片)

文件位置: `src/components/GlassCard.tsx`

```typescript
interface GlassCardProps {
  children: React.ReactNode;
  className?: string;
  hover?: boolean;     // 默认 true
  onClick?: () => void;
  id?: string;
}
```

样式特征:

- `glass-card` CSS 类: `bg-white/5 + backdrop-blur + border`
- 悬浮效果: `hover:scale-[1.01]` (微缩放)
- 可点击: onClick 存在时添加 `cursor-pointer` 和 `role="button"`, 支持键盘操作 (Enter/Space)

### 3.2 StatCard (统计卡片)

文件位置: `src/components/StatCard.tsx`

```typescript
interface StatCardProps {
  label: string;
  value: string;
  trend?: "up" | "down" | "neutral";
  className?: string;
}
```

- 基于 GlassCard 构建
- 标签: `text-xs text-text-secondary`
- 数值: `text-xl font-semibold font-num`
- 趋势色: up = `text-profit`, down = `text-loss`, neutral = `text-text-primary`

### 3.3 EmptyState (空状态)

文件位置: `src/components/EmptyState.tsx`

```typescript
interface EmptyStateProps {
  icon: React.ElementType;
  title: string;
  description: string;
  actions?: Array<{ label: string; onClick: () => void; variant?: "default" | "outline" }>;
}
```

布局:

- 垂直居中, py-12
- 图标: 48x48, `text-text-muted`
- 标题: `text-base font-medium text-text-primary`
- 描述: `text-sm text-text-secondary`
- 操作按钮: 可选, 水平排列

### 3.4 PageTransition (页面过渡)

文件位置: `src/components/PageTransition.tsx`

- 使用 framer-motion `motion.div`
- 进入动画: opacity 0->1, y 8->0, duration 200ms
- 包裹所有页面组件

### 3.5 ConfirmDialog (确认对话框)

文件位置: `src/components/ConfirmDialog.tsx`

```typescript
interface ConfirmDialogProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  title: string;
  description: string;
  confirmLabel?: string;    // 默认 "确认"
  cancelLabel?: string;     // 默认 "取消"
  variant?: "default" | "destructive";
  onConfirm: () => void;
  loading?: boolean;
}
```

基于 Radix UI AlertDialog:

- destructive 变体: 确认按钮红色
- loading 状态: 确认按钮显示 Loader2 旋转图标, 取消按钮禁用

### 3.6 Button (按钮)

文件位置: `src/components/ui/Button.tsx`

使用 class-variance-authority (cva) 管理变体:

**variant:**

| 变体         | 样式                                        |
|-------------|---------------------------------------------|
| default     | 蓝色背景, 白色文字                            |
| primary     | 蓝色背景, 白色文字                            |
| secondary   | 悬浮背景, 主文字色                            |
| danger      | 红色背景, 白色文字                            |
| destructive | 红色背景, 白色文字                            |
| ghost       | 次要文字色, 悬浮显示背景                       |
| outline     | 边框, 主文字色, 悬浮显示背景                   |

**size:**

| 尺寸     | 高度  | 水平内边距 | 字号    |
|---------|-------|-----------|---------|
| xs      | 24px  | 8px       | 11px    |
| sm      | 32px  | 12px      | 12px    |
| default | 36px  | 16px      | 14px    |
| md      | 36px  | 16px      | 14px    |
| lg      | 40px  | 24px      | 16px    |
| icon    | 36px  | -         | -       |
| icon-xs | 24px  | -         | -       |

特殊用法: Lab 页面中 "验证规律" 按钮使用内联样式覆盖为琥珀色:

```
className="gap-1.5 bg-accent-gold text-black hover:bg-accent-gold/90"
```

### 3.7 Badge (徽章)

文件位置: `src/components/ui/badge.tsx`

## 6. 通知系统

使用 **sonner** 库:

- 主题: dark
- 位置: top-right
- 宽度: 360px
- 容器类名: `glass-card` (毛玻璃效果)

典型用法:

```typescript
toast.success("操作成功");
toast.error("操作失败: " + err.message);
toast.info("提示信息");
```

自定义: 回测失败使用 `duration: Infinity`, 不自动消失。

## 11. 设计约束

1. 全局深色主题, 不支持亮色主题切换
2. 强调色统一使用琥珀色 (amber-400), 不使用系统蓝色作为主强调色
3. 盈亏色固定: 绿色 (#00C853) = 盈利/上涨, 红色 (#FF1744) = 亏损/下跌
4. 所有数值显示使用等宽字体 (font-mono), 确保数字对齐
5. 图标统一使用 Lucide React, 不混用其他图标库
6. 基础 UI 组件基于 Radix UI 原语, 确保无障碍访问
7. 不使用 emoji, 所有视觉元素通过图标或文字表达
8. 卡片悬浮效果统一使用 scale(1.01), 不使用阴影变化
9. 按钮禁用态: `opacity: 0.5 + pointer-events: none`
10. 表单输入框聚焦态: 琥珀色边框 (`focus:border-accent-gold`)
